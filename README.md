## 🚀 Быстрый старт (Docker)

Для работы с проектом используется **Docker** и **Makefile**.

### 1. Настройка переменнных окружения
Скопируйте пример конфига и установите свои параметры:
```bash
cp .env.example .env
```

### 2. Запуск базы данных
Поднимите контейнер с PostgreSQL 18.0:
```bash
make up
```

### 3. Полезные команды
- `make stop` — временно остановить базу (без удаления данных)
- `make restart` — временно остановить базу (применяет изменения в .env)
- `make logs` — посмотреть логи (полезно при ошибках подключения)
- `make clean` — удаляет всё: контейнеры и данные в таблицах
